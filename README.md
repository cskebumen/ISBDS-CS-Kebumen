# SIM ISBDS Cipta Sejati Cabang Kebumen

Sistem Informasi Manajemen untuk Institut Seni Bela Diri Silat (ISBDS) Cipta Sejati Cabang Kebumen. Aplikasi ini mengelola data anggota, riwayat sabuk, sertifikat, KTA, surat, dan log aktivitas.

## Teknologi
- **Frontend**: HTML, Tailwind CSS, JavaScript (ES6)
- **Backend & Database**: Supabase (PostgreSQL, Authentication, Storage)
- **Deployment**: Vercel (static site)

## Fitur
- Login dengan autentikasi Supabase
- Dashboard statistik anggota dan aktivitas
- Manajemen data induk anggota (CRUD, import/export Excel)
- Detail profil anggota dengan riwayat sabuk
- Penerbitan e-sertifikat dan e-KTA (digital)
- Manajemen e-surat (surat masuk/keluar)
- Log aktivitas real-time

## Prasyarat
- Akun Supabase (https://supabase.com)
- Akun Vercel (https://vercel.com)
- Node.js (opsional untuk development lokal)
